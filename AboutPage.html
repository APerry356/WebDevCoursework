THE WORLD AFTER

So What Is This Game?

This game is all about surviving a war torn land and reaching an end goal based on your selected charecter. You will be presented with scenarios and will have to decide from a list of set options how to deal with them. It is possible to fail the gane if you make a very wrong desition in a dangerous scenario. Each scenario will have enough infomation to help you decide which are the correct options. An option might be correct, but it might be a bad choice with negative outcomes for other charecters in the game.
